# 经济时间序列课程资料说明

## 组织规则

本目录按“每一章一个独立文件夹”的方式组织。每章文件夹和主 Markdown 文件都使用中文命名。

- `第2章_平稳时间序列模型/`

- `第1章_差分方程/第1章_差分方程_教材式中文讲义.md`

这是第 1 章当前主要维护文件。修改正文时，优先修改这一份 Markdown。

## 第 2 章主文件

- `第2章_平稳时间序列模型/第2章_平稳时间序列模型_教材式中文讲义.md`

这是第 2 章当前主要维护文件。大纲文件保留在同一章文件夹中，供调整结构时参考。

## 每章文件夹内部结构

- `图片/`：本章讲义使用的图片。每张图单独一个子文件夹，并附说明文件，说明来源、用途和引用文档。
- `导出文件/`：由 Markdown 或脚本导出的 PDF、Word、PPT、HTML 等文件。
- `脚本/`：生成 PDF、Word 或处理 Markdown 的脚本。
- `草稿/`：早期版本、备份文件和不再作为主版本维护的材料。
- `质检/`：PDF、Word、PPT 等文件的渲染检查图或检查记录。

## 使用建议

平时编辑和备课时，以各章文件夹中的 Markdown 为准；需要发给学生或课堂展示时，再使用该章 `导出文件/` 中的 PDF、Word 或 PPT。

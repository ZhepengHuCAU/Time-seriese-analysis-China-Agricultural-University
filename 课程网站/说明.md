# 课程网站说明

## 用途

本文件夹用于生成 GitHub Pages 课程网站。

## 当前结构

- `mkdocs.yml`：网站配置文件。
- `docs/`：网站正文内容。
- `docs/index.md`：课程首页。
- `docs/第1章_差分方程/index.md`：第 1 章网页内容。
- `docs/第2章_平稳时间序列模型/index.md`：第 2 章网页内容。

## 发布方式

GitHub Actions 会在推送到 GitHub 后自动安装 MkDocs Material，并发布到 GitHub Pages。
